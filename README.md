# 🎓 EduSync - School Management System (Sign Up / Sign In)

## 📁 Structure du projet

Le projet est organisé en plusieurs dossiers pour une meilleure gestion du code :

### 📂 public/
Contient les fichiers accessibles par l'utilisateur :

- `index.php` : Page d'accueil  
- `login.php` : Page de connexion  
- `register.php` : Page d'inscription  
- `dashboard.php` : Tableau de bord après connexion  

---

### 📂 includes/
Contient les fichiers réutilisables :

- `header.php` : En-tête du site  
- `footer.php` : Pied de page  
- `functions.php` : Fonctions utiles du projet  

---

### 📂 scripts/
Contient les scripts liés à l'authentification :

- `authprocess.php` : Traitement de connexion / inscription  
- `logout.php` : Déconnexion de l'utilisateur  

---

## ⚙️ Technologies utilisées

- PHP  
- HTML / CSS  
- (Optionnel : Tailwind CSS)

---

## 🚀 Objectif du projet

Créer un système de gestion scolaire simple avec :
- Inscription des utilisateurs  
- Connexion sécurisée  
- Gestion de session  
- Accès à un tableau de bord
